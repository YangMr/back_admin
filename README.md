##### 4.5.1 在src目录下创建.env.development文件

- 配置代理的名称

    ```
    VUE_APP_BASE_API = "开发模式下的代理名称(/dev-api)" 
    ```

    

- 配置代理的地址

    ```
    VUE_APP_SERVICE_URL = "要代理的接口地址"
    ```
    
    

##### 4.5.2 在src目录下创建.env.production文件

- 配置代理的名称

    ```
    VUE_APP_BASE_API = "生产模式下的代理名称(/pro-api)"
    ```
